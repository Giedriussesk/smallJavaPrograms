# Small Java Programs

A collection of console-based applications built to practice Java fundamentals, logic flow, and mathematical operations.

## Project List
- **Atmproject.java:** A banking menu implementation using switch-case logic for account management.
- **Calculator.java:** Basic arithmetic operations using conditional statements.
- **Compoundcalculator.java:** Financial calculations utilizing the `Math` library.
- **Diceroll.java:** Implementation of the `Random` class with ASCII art output.
- **MadLibs.java:** String manipulation and user input handling via the `Scanner` class.
- **Numberguessing.java:** A game loop utilizing `do-while` structures.
- **Quizgame.java:** Array handling and logic blocks for question-response verification.
- **Shoppingcart.java:** Practical application of price accumulation and item management.
- **Tempconversion.java / Weightconversion.java:** Unit conversion tools using standard formulas.
- **MortgageCalculator.java:** Advanced math-based project focusing on interest rate formulas.

## Concepts Practiced
- **Input Handling:** Managing different data types (int, double, String) using `java.util.Scanner`.
- **Control Flow:** Implementation of `if-else`, `switch`, and various loop structures (`for`, `while`, `do-while`).
- **Standard Libraries:** Utilization of `java.util.Random` and `java.lang.Math`.
- **Method Structure:** Organizing code into static methods for improved readability and modularity.
