# Small Java Programs

This is a collection of small console-based apps I built while teaching myself the fundamentals of Java. I used these projects to move past basic syntax and actually practice logic flow, user input, and math operations.

## 🕹️ Project List

* **Atmproject.java:** A simple banking menu using `switch` cases to handle deposits, withdrawals, and balance checks.
* **Calculator.java:** A classic calculator for basic arithmetic using conditional logic.
* **Compoundcalculator.java:** Practice with the `Math` library to calculate compound interest over time.
* **Diceroll.java:** Uses the `Random` class and ASCII art to simulate rolling dice.
* **MadLibs.java:** A fun way to practice string manipulation and capturing different types of user input.
* **Numberguessing.java:** A game that uses a `do-while` loop to let the user guess a random number.
* **Quizgame.java:** Practice using arrays or logic blocks to check user answers against a set of questions.
* **Shoppingcart.java:** Simple logic for adding up prices and managing a list of items.
* **Tempconversion.java / Weightconversion.java:** Utility tools for converting units (Celsius to Fahrenheit, etc.) using standard formulas.
* **MortgageCalculator.java:** A more advanced math-based project using interest rate formulas and user-defined time periods.

## 🧠 Concepts I Practiced
* **Input Handling:** Getting comfortable with the `Scanner` class and handling different data types (int, double, String).
* **Control Flow:** Using `if-else`, `switch` statements, and various loops (`for`, `while`, `do-while`).
* **Java Libraries:** Learning how to import and use `java.util.Scanner`, `java.util.Random`, and `java.lang.Math`.
* **Method Structure:** Organizing code into static methods to keep the `main` method clean.
